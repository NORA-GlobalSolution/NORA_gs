# NORA - Nutrient Optimization and Retention Assistant

Sobre o Projeto

A NORA (Nutrient Optimization and Retention Assistant) é uma cápsula inteligente desenvolvida para armazenar e preservar alimentos durante missões espaciais de longa duração. O projeto busca garantir que os nutrientes dos alimentos sejam mantidos por mais tempo por meio do monitoramento e controle das condições de armazenamento.

Esta landing page foi criada para apresentar o conceito da NORA, seus objetivos, funcionalidades e benefícios para futuras missões espaciais.

---

## Objetivo

Desenvolver uma solução tecnológica capaz de otimizar a conservação de alimentos em ambientes espaciais, reduzindo perdas nutricionais e contribuindo para a saúde e o bem-estar dos astronautas.
